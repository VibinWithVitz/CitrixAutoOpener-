This is a script for Mac that will open all relevant radiation oncology applications from SCMC. The user will still need to use their phone's 2FA. All passwords are stored securely using macOS' encrypted Keychain.

## Installation

1. Unzip the folder and put it somewhere permanent (e.g. your Documents folder)
2. Double-click **Install CitrixOpener.command**
3. Follow the on-screen prompts — it installs everything automatically
4. When finished, go to **System Settings → Keyboard → Keyboard Shortcuts → Services**, find **Citrix Auto-Login**, and assign a shortcut (e.g. Ctrl+Shift+C)

That's it! Press your shortcut anytime to auto-launch your Citrix apps.
